# Media — drop zone

Drop demo files here and I'll embed them into the parent project README at the right place.

## What to drop, where

| Filename | Goes into README section | Recommended format |
|---|---|---|
| `architecture.png` | Top of the README (replaces the rendered Mermaid) | PNG, ≤ 1600 px wide |
| `demo-landscape-sculpt.gif` | "What it is" section | GIF, ≤ 8 MB, 600–1000 px wide |
| `demo-mask-paint.gif` | Architecture diagram caption | GIF or short MP4 |
| `demo-sequencer-keyframe.gif` | "Custom additions" section | GIF or short MP4 |
| `ide-toolcall-trace.png` | "Engineering writeup → Architecture choices" | PNG screenshot |

## Capture tips

- GIFs > 8 MB get slow to load on GitHub. Use [Gifski](https://gif.ski) or ScreenToGif's optimize tool to compress.
- For longer clips, drop an MP4 instead — GitHub will inline a video player.
- Screenshots: prefer dark mode for the editor (matches the kit's dark UI), and crop to the relevant panel — full-screen UE screenshots are too noisy.
- If the GIF needs a caption, drop a sibling `architecture.md` or similar with the caption text and I'll embed it.

Once dropped, ping me and I'll edit the parent README to reference each file.
